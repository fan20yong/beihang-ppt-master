# Beihang Native PPTX Master Assets

This directory contains privacy-safe, master-only PPTX files derived from local Beihang template decks.

Included content:
- Slide masters
- Slide layouts
- Themes
- Template media referenced by masters/layouts
- One blank placeholder slide required for PPTX validity

Removed content:
- Original source slides
- Speaker notes
- Comments and comment authors
- Document properties and custom XML
- Embedded files
- Charts, diagrams, tags, handout masters, and printer settings

Use these files when a task needs native PowerPoint layout inheritance or the `template-fill` workflow. For the default SVG-generation pipeline, prefer:

`templates/decks/beihang_university/`
